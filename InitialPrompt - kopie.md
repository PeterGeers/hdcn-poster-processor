HDCN Poster Processor is an reactJS app using openrouter.ai / Gemini pro/ Google lens to

1. Upload a poster (from the computer or from Google drive)
2. Retrieve the text from the poster and show the text retrieved in a window
   2.1 Change the date of the poster to the date retrieved from the text
   2.2 Let the user approve the date update
   2.3 Upload the poster to the public folder "PostersForAgenda" so it can be found by anyone, If it already exists do not store it.
   2.4 Provide the url to access the poster (preferably in lh3 format) to the next step
3. Make a Agenda posting in one of the following (Nationaal, Internationaal en Beurzen en Diversen) google agenda's from webmaster@h-dcn.nl
   3.1 The user should have the opportunity to modify the agenda before posting and add the start and end date
   3.2 The url of the poster should be attached to the booking as a google drive element
4. Upload the poster to Google photos from webmaster@h-dcn.nl
   4.1 Ensure the date of the poster is set correctly to the same start date as used for the agenda
   4.2 Add the poster to the album "Flyer / Poster Verzameling"
   5 Use of a backend with google api's for drive, calendar and google photos is recommended as this may increase the level of automation
